订单详细接口
url：/order/info

请求参数

字段 必选 类型及范围 说明
order_id int 订单id

响应参数

字段 必选 类型及范围 说明
->data->order_id int 订单id
->data->order_sn int order_sn
->data->order_status int 订单状态
->data->order_status_txt string 订单状态文本
->data->order_time string 订单状态最后变更时间戳
->data->goods array 订单商品信息
->data->goods->order_id int order_id
->data->goods->goods_name string goods_name
->data->goods->price int price
->data->goods->quantity int quantity
->data->goods->goods_image string goods_image
->data->coupon array 订单优惠券
->data->coupon->title string 优惠标题
->data->coupon->amount float 优惠金额（正式）
->data->coupon->money float 优惠面额

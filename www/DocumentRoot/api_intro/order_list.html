订单列表接口
url：/order/orders

请求参数

字段 必选 类型及范围 说明
status string unpay：未付款 payed 待发货 shipped：已发货 finished：已完成 closed：已关闭
page int 翻页

响应参数

字段 必选 类型及范围 说明
->data->count int 总订单数
->data->list array 订单列表
->data->list->order_id int 订单id
->data->list->order_sn int order_sn
->data->list->order_status_txt int order_status_txt 订单状态文本
->data->list->order_status_alert int 订单状态提醒
->data->list->payment_code string payment_code 订单支付方式 alipay : appalipay
->data->list->goods_name int goods_name
->data->list->goods_id int id
->data->list->price int 商品单价
->data->list->quantity int 购买数量
->data->list->goods_image string 图片地址

API  sms/login

为用户生成一个新的验证码，并发送短信通知对应用户

url

域名/api/sms/login    如 http://api.ymall.com/sms/login

支持格式

json

http请求方式

get

是否需要登录

是

 

请求参数

 

必选

类型及范围

说明

user_name

true

string

手机号码

type

true

string

请求类型 type: login, reg, repwd

注意事项

1.  可否在客户端限制每分钟只能请求一次

2. 用户申请验证码后，如未通过验证，则验证码保持不变

订单退款接口
url：/refund/apply

请求参数

字段 必选 类型及范围 说明
order_id 必填 int 订单 order_id
card_no 必填 string 支付宝账号
refund_desc 非 string 退款申请事由

响应参数

字段 必选 类型及范围 说明
status int 200
data->refund_id int 退款id
error int 5000 请完善退款资料
error int 5001 只能对自己订单申请退款
error int 5002 已经提交退款申请，请等待客服处理
订单退款详细接口
url：/refund/info

请求参数

字段 必选 类型及范围 说明
order_id 必填 int 订单 order_id

响应参数

字段 必选 类型及范围 说明
status int 200
data->refund_id int 退款id
card_no string 退款支付宝卡号
refund_desc string 退款申请描述

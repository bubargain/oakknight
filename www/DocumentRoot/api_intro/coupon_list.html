订单列表接口
url：/coupon/index

请求参数

字段 必选 类型及范围 说明
state string history 过期; valid：有效（默认）
page int 翻页

响应参数

字段 必选 类型及范围 说明
->data->count int 总订单数
->data->list array 列表
->data->list->title string 优惠券标题/td>
->data->list->coupon_sn string 优惠券编码
->data->coupons->min_order_amount float 优惠券使用最低金额
->data->coupons->coupon_text string 优惠券使用说明
->data->list->amount float 优惠券金额
->data->list->state int 订单状态 0：未开启; 1：进行中; 2：已使用未支付; 3：已使用并支付;4：已结束（未使用）; 有效：0，1； 历史：2，3，4
->data->list->state_str string 订单状态说明
->data->list->from_time int 起始时间
->data->list->end_time int 起始时间
